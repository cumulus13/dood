[cumulus13](mailto:cumulus13@gmail.com)

## usage

```bash:
Usage: dood [-h] [-d] [-p PATH] [-n SAVE_AS] URL

Positional Arguments:
  URL                   url generate to

Options:
  -h, --help            show this help message and exit
  -d, --download        Download it
  -p, --path PATH       Save download to directory
  -n, --save-as SAVE_AS
                        Save as
```
